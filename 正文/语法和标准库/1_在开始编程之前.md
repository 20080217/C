## 4. 关于C

### 3.1 发展历史

1. 机器码
2. 汇编
3. ALGOL CPL BCPL
4. C

### 3.2 标准

1. ANSI C（被ISO采纳）
2. 从c89到c23（建议大家有空了看看标准）

### 3.3 编译过程

1. 8个翻译阶段
2. **预处理器**（在翻译阶段4）
3. **编译器**（在翻译阶段7）
4. **汇编器**
5. **链接器**（在翻译阶段8）

### 3.4 应用领域

1. 系统编程：操作系统，编译器，数据库等
2. 嵌入式开发，IoT
3. 网络编程
4. 图形图像处理

### 3.5 推荐网站

1. [cppreference](https://zh.cppreference.com)（学习 C 和 C++ 必备网站）
2. [compiler explorer](https://godbolt.org/)（在线编译器，支持**生成汇编**等多项功能，支持非常多的语言和多种编译器）

### 3.6 推荐的C书籍
